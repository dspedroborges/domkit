domkit / tabs component

built with domkit

A tabs component wired entirely through signals and DOM helpers — no framework, no build step. The active tab is reactive state; everything else follows.

// active tab as a signal const active = domkit.signal('overview'); // switch tab on click domkit.on('#tab-list', 'click', e => { const btn = e.target.closest('[data-tab]'); if (btn) active.set(btn.dataset.tab); }); // react to changes domkit.effect(() => syncUI(active.get()), [active]);
